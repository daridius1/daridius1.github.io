Bueno, al final lo hice. ¿Qué tan difícil fue? Poco. La idea estaba bastante clara desde hace más de un año y, con la ayuda del clanker, fue bastante sencillo de completar. Diría que la mayor paja fue sacar la respectiva API key para cada plataforma.

¿Qué se terminó utilizando?

  • Google Drive: Para el formulario y la API que permite leerlo.
  • API de Gemini: Que está gratis, para censurar las confesiones.
  • Pillow: Para dibujar la wea (la imagen de la confesión).
  • API de imgbb: Para subir la imagen, ya que la API de IG pide que la imagen sea pública.
  • API de Instagram: Para publicar la confesión.
  • Raspberry de Blaz: Donde corre todo el sistema.

¿Cómo creo que le va a ir?

Ni idea. En vacaciones de verano yo creo que nadie está ni ahí con confesar weas, así que no imagino mucha actividad los primeros días. Pero onda, en marzo y cosas así, cuando haya alta demanda de cahuines y confesiones, siento que la disponibilidad para publicar que tiene un bot frente a un ser humano es demasiado superior y va a funcionar bien.

Debo admitir que lo de las confesiones no me prende tanto como el automatizar algo que era estúpido que lo hiciera un humano; eso sí que me prende. Y creo que el tema de las confesiones es un perfecto ejemplo de algo que, aunque tiene sus pifias, ofrece ventajas respecto a un wn haciéndolo a mano que son ridículamente altas.
